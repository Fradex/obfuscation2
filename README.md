# obfuscation2

Консольный инструмент для обфускации C# решений на базе изменения IL и последующей декомпиляции в «нечитабельный» исходный код.

## Возможности
- Принимает путь до `.sln` и каталог вывода.
- Собирает решение, модифицирует IL (вставляет непрозрачный предикат), затем декомпилирует обратно в C#.
- Сохраняет сигнатуры методов и корректность исполнения.

## Сборка single-file

```bash
./scripts/publish_single_file.sh linux-x64
```

Готовый исполняемый файл будет в каталоге `dist`.

## Применение

```bash
./dist/ObfuscationTool --sln /path/to/solution.sln --out /path/to/output
```

Результат:
- `output/assemblies` — обфусцированные сборки.
- `output/sources/<AssemblyName>` — декомпилированный C# исходный код после вмешательства в IL.

## Локальная сборка (без single-file)

```bash
dotnet build src/ObfuscationTool/ObfuscationTool.csproj -c Release
```

## Примечания
- Инструмент модифицирует IL, добавляя непрозрачный предикат в начало каждого метода.
- Для корректного вывода требуется установленный .NET SDK 8.0.
